#***OBS***:
**Antes de executar o projeto você deve digitar no terminal do editor de código (ex: VScode) os seguintes comandos**:

----------------------------------------------------
> `npm init -y`                                   --
> `npm i whatsapp-web.js`                         --
> `npm qrcode-terminal`                           --
----------------------------------------------------
Isso instalará as dependencias necessárias para o programa funcionar.
depois no terminal do projeto digite:

> `npm start`

assim o projeto inicializará e irá gerar um Qrcode para ser escaneado pelo WhatsApp na opção **conectar dispositivo**:
Após isso basta entrar em qualquer mensagem e enviar **INICIAR** que o bot começará a responder.
